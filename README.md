Preview [PostCSS] output in [Atom]
==================================

Atom [source-preview] provider for [PostCSS].

Requirements
------------
* [source-preview]
* [language-postcss]

Install
-------
`apm install source-preview-postcss` or search "postcss" under packages within Atom.
